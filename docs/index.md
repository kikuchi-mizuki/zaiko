# 追加発注・在庫照会システム

## プロジェクト概要

高島屋15店舗向けの追加発注・在庫照会・出荷依頼を自動化するシステムの要件定義書です。

### 目的

繁忙期における突発的な追加発注対応を効率化し、以下を実現します：

- 店舗がLINEで即時に在庫可否を確認できる
- 本部・貴社・加工会社間の伝言ゲームを削減する
- 在庫の確度を管理し事故を防ぐ
- 出荷依頼・納品書生成を自動化する

### システム構成

- **フロント**: LINE Official Account
- **バックエンド**: Python (FastAPI)
- **データベース**: Supabase (PostgreSQL)
- **ストレージ**: Supabase Storage
- **帳票**: PDF自動生成
